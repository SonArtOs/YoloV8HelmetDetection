Модель YoloV8 на детекцию строительных касок. Использовался датасет с Kaggle. 
Датасет состоял из 5000 фотографий людей как с касками, так и без касок для более лучшего качества. 
По результатам можно сказать, что модель хорошо детектит шлемы. 
Результаты обучения показывают mAP50 = 0.62 и mAP50-95 = 0.36. Модель стабильно обучается, loss снижается, precision/recall сбалансированы. 
Для демонстрации касок на видео качество достаточное, но для промышленного использования нужно больше данных и тюнинг.
